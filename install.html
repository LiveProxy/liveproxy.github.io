Version

Installing

Latest release (pip)

 
# python3 -m pip install -U liveproxy

Development version (pip)

 
# python3 -m pip install -U git+https://github.com/back-to/liveproxy.git

Development version (git)

 
$ git clone git://github.com/back-to/liveproxy.git
$ cd liveproxy
# python3 setup.py install
